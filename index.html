What is console log?


The console. log() method outputs a message to the web console. The message may be
a single string (with optional substitution values), or it may be any one or more JavaScript
objects. Note: This feature is available in Web Workers


What is visule studio?

Visual Studio Code is a source-code editor made by Microsoft for Windows, Linux and macOS.
Features include support for debugging, syntax highlighting, intelligent code completion,
snippets, code refactoring, and embedded Git.
